• современный дизайн
  • 2 режима: простой, расширенный
  • вывод результата с адаптацией под размер окна
    • форматирование с экспаненципальной приставкой
    • подбор размера шрифта под размер окна
  • Используемые технологии:
    • Javascript
    • Angular
    • CSS
    • Html
  • Исправляет ввод:
    • нескольких десятичных точек подряд на одну в каждом числе
    • нескольких операторов подряд
    • незакрытых скобок
    • скобок без оператора, добавляет оператор умножения перед и после скобок
    • цифры ноль в начале числа
  • Обрабатывает ошибки:
    • деления на ноль
  • Возможность расширения кода (разделение логики калькулятора от представления)
  • Код читабелен
